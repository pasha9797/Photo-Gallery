# Photo Gallery
Приложение поддерживает следующие функции:
1. Скроллинг фотографий с описаниями.
1. Лайки фотографий. (для авторизованных пользователей)
1. Загрузка новых фото. (для авторизованных пользователей)
1. Личный профиль.
